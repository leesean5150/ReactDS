## Getting started

### Clone repository

```shell
https://github.com/leesean5150/ReactDS.git
```

### Install typescript

```shell
npm install -g typescript
```

### Install dependencies

```shell
npm install
```

### Run

```shell
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
